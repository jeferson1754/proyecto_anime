### 🚀 Visión General
Este sistema es una aplicación web integral diseñada para la organización, seguimiento y archivo de series y películas de anime. La plataforma gestiona el ciclo de vida completo del contenido, permitiendo a los usuarios administrar su progreso de visualización, calendarios de emisión y colecciones multimedia de forma centralizada.

---

## 🛠️ Capacidades del Sistema
El núcleo de la aplicación se basa en una gestión de estados dinámica:
* **Gestión de Emisión:** Seguimiento en tiempo real de series activas y calendarios de estreno.
* **Control de Pendientes:** Organización de listas de seguimiento futuras y planificación de contenido.
* **Archivo Histórico:** Registro detallado de series finalizadas con estadísticas de completitud.
* **Gestión Multimedia:** Módulo especializado para la administración de bandas sonoras (Openings/Endings) y metadatos asociados.
* **Sistema de Clasificación:** Algoritmo de valoraciones personalizadas para el análisis de calidad del contenido visto.

## ✨ Características Técnicas
* **Flujos de Trabajo por Estado:** Lógica de negocio específica para cada etapa del contenido (Pendiente → En Emisión → Finalizado).
* **Operaciones CRUD Avanzadas:** Interfaz robusta para la creación, lectura, actualización y eliminación de registros de anime.
* **Programación de Emisiones:** Módulo de calendario para la gestión de lanzamientos periódicos.
* **Infraestructura Modular:** Arquitectura diseñada para separar la gestión de contenido de la lógica de clasificación.

## 🛠️ Stack Tecnológico
* **Backend:** PHP (Gestión de lógica de estados y flujos de trabajo).
* **Base de Datos:** MySQL (Relaciones complejas entre series, temporadas y contenido multimedia).
* **Frontend:** HTML5, CSS3, JavaScript (Interfaz enfocada en la experiencia de usuario y organización visual).
